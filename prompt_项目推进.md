## 1. 实现瞬态仿真，采用完全隐式方法，并支持自适应步长开关。

仔细设计瞬态仿真器在项目体系中的位置，最终实现以下复杂度的流程：

```bash
时间步进
{
   picard迭代（电热双向耦合）
   {
       根据上一步温度场 T 更新材料属性电导率

       电 -> 热 单向耦合
       {
          1. 电场求解器求解电势分布 V
          2. 将电势分布 V 传递给热场求解器
          3. 热场求解器根据电势分布 V 计算焦耳热源项 Q
          4. 热场求解器更新温度场 T
       }
   }

   得到该时间步的温度场 T 和电势场 V

   热 -> 力 单向耦合
   {
       1. 热场求解器求解温度分布 T
       2. 将温度分布 T 传递给力学求解器
       3. 力学求解器根据温度分布 T 计算热膨胀引起的应变
       4. 力学求解器计算位移场 U
   }
}
```
