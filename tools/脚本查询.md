# 工具脚本说明

---

## 1. compare_solution_txt.py — FEM 结果对比脚本

### 功能
比较一份 FEM 导出的 txt 结果与一份 COMSOL 导出的 txt 结果，按坐标匹配计算误差指标。
支持稳态（标量/矢量自动检测）和瞬态模式。

### 用法

```bash
python3 tools/compare_solution_txt.py --mine <FEM结果> --comsol <COMSOL参考> [选项]
```

### 参数表

| 参数                | 类型   | 默认值   | 说明                                                               |
| ------------------- | ------ | -------- | ------------------------------------------------------------------ |
| `--mine`            | 路径   | （必填） | FEM 求解器导出的 txt 文件路径。若不存在，自动尝试 `results/` 前缀  |
| `--comsol`          | 路径   | （必填） | COMSOL 导出的参考 txt 文件路径                                     |
| `--value-col`       | 整数   | 自动检测 | 指定比较的值列索引（0-based），-1 表示最后一列。指定后强制标量模式 |
| `--transient`       | 开关   | 关       | 启用瞬态模式：按 `(fields_per_step × n_steps)` 解析值列            |
| `--fields-per-step` | 整数   | 3        | 瞬态模式下每时间步的场数（默认 3: V, T, disp）                     |
| `--field-names`     | 字符串 | V,T,disp | 瞬态模式下各场名称，逗号分隔                                       |
| `--tol`             | 浮点   | 1e-8     | 坐标匹配的量化容差（用于空间哈希分桶）                             |

### 输出指标

| 指标    | 含义                   |
| ------- | ---------------------- |
| MAE     | 平均绝对误差           |
| RMSE    | 均方根误差             |
| MaxAE   | 最大绝对误差           |
| MeanRel | 平均相对误差           |
| matched | 成功匹配的坐标点数     |
| miss    | 未匹配到参考坐标的点数 |

### 示例

**稳态标量场**（温度）：
```bash
python3 tools/compare_solution_txt.py \
  --mine results/busbar_oneway/temperature.txt \
  --comsol results/busbar_oneway_temperature.txt
```

**稳态矢量场**（位移，自动检测）：
```bash
python3 tools/compare_solution_txt.py \
  --mine results/bipolar_thermo_mechanical/displacement.txt \
  --comsol results/bipolar_displacement.txt
```

**稳态标量场（指定列）**：
```bash
python3 tools/compare_solution_txt.py \
  --mine results/bipolar_thermo_mechanical/displacement.txt \
  --comsol results/bipolar_displacement.txt \
  --value-col -1 --tol 1e-8
```

**瞬态多场**：
```bash
python3 tools/compare_solution_txt.py \
  --mine results/busbar_transient/transient.txt \
  --comsol results/bubar_transient.txt \
  --transient --fields-per-step 3 --field-names "V,T,disp"
```

**瞬态 VBC**：
```bash
python3 tools/compare_solution_txt.py \
  --mine results/busbar_transient_vbc/transient.txt \
  --comsol results/bubar_transient_vbc.txt \
  --transient --fields-per-step 3 --field-names "V,T,disp"
```

### 文件格式要求

- 每行格式：`x y z val1 [val2 val3 ...]`
- 以 `%` 或 `#` 开头的行为注释，自动跳过
- 坐标和值之间用空格或逗号分隔
- 稳态标量：至少 4 列（x, y, z, value）
- 稳态矢量：至少 6 列（x, y, z, ux, uy, uz）
- 瞬态：值列数 = fields_per_step × n_steps

## 2. mphtxt_to_msh.py — COMSOL 网格转换脚本

### 功能
将 COMSOL 导出的 `.mphtxt` 网格文件转换为 MFEM 可读取的 `.msh` (Gmsh) 格式。

### 用法

```bash
python3 tools/mphtxt_to_msh.py <输入.mphtxt>
```

### 示例

```bash
python3 tools/mphtxt_to_msh.py mesh_data/bipolar_plate.mphtxt
# 输出: mesh_data/bipolar_plate.msh
```

## 3. compare_time_curves.py — 瞬态采样点曲线对比脚本

### 功能

比较本项目 `transient.txt` 与 COMSOL 瞬态宽表在采样点上的时间曲线。支持随机抽取 N 个可匹配点，也支持显式指定点或从文件读取点。

### 用法

```bash
python3 tools/compare_time_curves.py --mine <FEM瞬态结果> --comsol <COMSOL参考> [选项]
```

### 参数表

| 参数 | 类型 | 默认值 | 说明 |
| ---- | ---- | ------ | ---- |
| `--mine` | 路径 | （必填） | 本项目导出的 `transient.txt` |
| `--comsol` | 路径 | （必填） | COMSOL 导出的瞬态参考 txt |
| `--random-points` | 整数 | 5 | 未指定点时，从可匹配坐标中随机抽样 N 个点 |
| `--seed` | 整数 | 12345 | 随机抽样种子 |
| `--points` | 字符串 | 空 | 显式点列表，如 `"x,y,z; x,y,z"` |
| `--point-file` | 路径 | 空 | 每行一个采样点，支持空格或逗号分隔 |
| `--fields-per-step` | 整数 | 3 | 每个时间步的场数 |
| `--field-names` | 字符串 | V,T,disp | 字段名，逗号分隔 |
| `--fields` | 字符串 | 全部 | 只比较部分字段，可填字段名或 0-based 索引 |
| `--tol` | 浮点 | 1e-8 | 坐标匹配容差 |
| `--output-csv` | 路径 | 空 | 写出逐点逐时间步误差明细 |
| `--plot-dir` | 路径 | 空 | 输出 FEM 与 COMSOL 时间曲线对比图片目录 |
| `--plot-format` | 字符串 | svg | 图片格式，支持 `svg` 和 `png`；PNG 需要 matplotlib |

### 示例

**随机抽取 8 个点：**

```bash
python3 tools/compare_time_curves.py \
  --mine results/busbar_transient_vbc/transient.txt \
  --comsol results/busbar_transient_vbc.txt \
  --random-points 8 --seed 2026 \
  --fields-per-step 3 --field-names V,T,disp \
  --plot-dir results/busbar_transient_vbc/curve_plots
```

**指定点并导出图片和 CSV：**

```bash
python3 tools/compare_time_curves.py \
  --mine results/busbar_transient_vbc/transient.txt \
  --comsol results/busbar_transient_vbc.txt \
  --points "0.0975,0,0.1; 0.1,0,0.1" \
  --plot-dir results/busbar_transient_vbc/curve_plots \
  --output-csv results/curve_compare.csv
```
